# Direct Graph Engine
DirectGraphEngine is an experimental workflow engine written in Go that executes tasks using a directed graph (DAG / DCG) execution model.

The project is currently under active development and APIs may change without notice.

## Features
1. Directed graph workflow execution
2. Task dependency management
3. Conditional execution through expressions
4. Runtime state management
5. Pluggable connectors for data access and storage
6. Structured logging
